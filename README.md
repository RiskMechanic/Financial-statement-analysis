# Financial-statement-analysis
This repository contains cleaned, structured, and documented financial statements used across my academic journey.

Each dataset includes:
->standardized income statements, balance sheets, and cash‑flow statements
->data on source, year, and transformation steps
->ready‑to‑use formats for modeling, dashboards, and reproducible analysis

Built as part of my BSc in BA, this library demonstrates my approach to transparent methodology, data governance, financial diagnostics.
